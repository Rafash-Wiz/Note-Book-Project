# Note Book App

A simple React notes app built with Vite.

Users can:

- create notes with a title and note content
- edit existing notes
- delete notes
- favorite and unfavorite notes
- see the date and time each note was created

Favorited notes move to the top of the list. When unfavorited, they return to their original position.

## Project Folder

The main app lives in:

`Note-Book-App`

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Getting Started

From the project folder:

```bash
cd Note-Book-App
npm install
npm run dev
```

## Available Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Features

- centered note creation form
- note editing with save
- note deletion
- favorite toggle button
- creation timestamp on each note
- notes reordered based on favorite state
